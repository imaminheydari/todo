# ⚡ TaskFlow — To-Do List

A modern, sleek single-file to-do list app with a dark/light theme, built entirely in HTML, CSS, and JavaScript.

## Features

- **Add, edit, delete tasks** with a clean UI
- **Priority levels** — High, Medium, Low with color-coded indicators
- **Categories** — Work, Personal, Shopping, Health, Study, Other
- **Due dates** with overdue detection
- **Subtasks** — break tasks into smaller steps
- **Notes/descriptions** for each task
- **Drag & drop** reordering
- **Sorting** — by newest, oldest, priority, due date, or alphabetically
- **Filters** — All, Active, Completed with search
- **Dark/Light theme** toggle
- **Stats dashboard** — total, active, completed counts with progress bar
- **Undo delete** with toast notification
- **Import/Export** tasks as JSON
- **Keyboard shortcuts** for power users
- **LocalStorage** persistence — your data stays across sessions
- **Responsive design** — works on mobile and desktop

## Usage

Open `index.html` in any modern browser. No build step, no dependencies.
